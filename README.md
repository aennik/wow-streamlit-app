## Worum geht es?

Dieses Projekt ist im Rahmen meiner Weiterbildung entstanden und hat mir geholfen, den Aufbau einer kleinen interaktiven `Streamlit`-App zu üben.

Das Thema ist an **World of Warcraft** angelehnt.  
Der Fokus liegt dabei nicht auf einer fertigen Produkt-App, sondern auf dem Üben von App-Struktur, Design, Navigation und interaktiven Elementen.

---

## Was ich dabei geübt habe

- Aufbau einer mehrseitigen Streamlit-App
- Navigation zwischen verschiedenen Bereichen
- Arbeiten mit eigenen Design-Elementen
- Strukturierung von Python-Dateien
- Erste Dashboard-Elemente
- Umgang mit Daten und Anzeigeelementen

---

## Projektbestandteile

| Bereich | Inhalt |
|---|---|
| App-Aufbau | Grundstruktur einer Streamlit-App |
| Design | Eigene Farben, Layout-Ideen und visuelle Gestaltung |
| Navigation | Mehrere Seiten und klare Nutzerführung |
| Daten | Einbindung und Darstellung von Daten |
| Übung | Praktisches Weiterlernen mit Python und Streamlit |
